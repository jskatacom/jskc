Template Description:

The Healthy Blog template is a grat blog template for Bootstrap. It has the maxcdn links for bootstrap css and js files, as well as the jquery dependency, so all the resources are loaded from online sources.

Template Features:

  • Custom transparent navbar
  • Header background image
  • Creative use of Bootstrap 3 panels
  • A nice color scheme
