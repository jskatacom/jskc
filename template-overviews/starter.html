Template Description:

The starter template is the simplest possible Bootstrap template. Its main purpose is to speed up the setup of Bootstrap theme development, and to utilize the maxcdn links for bootstrap css and js files, as well as the jquery dependency.

Template Features:

  • Navbar from getbootstrap.com
  • Sample jumbotron element
  • css, jquery, and js files served by maxcdn
  • included blank custom.css file to speed up the setup time
