Template Description:

The Complex Nav Starter template's main purpose is to speed up the development of Bootstrap themes with complex navigation and to utilize the maxcdn links for bootstrap css and js files, as well as the jquery dependency.

Template Features:

  • Complex nav with additional elements
  • Clear markup
  • css, jquery, and js files served by maxcdn
  • included minimal custom.css file to speed up the setup time
