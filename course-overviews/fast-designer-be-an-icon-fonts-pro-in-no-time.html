Course Description:

In this class you'll get to see excellent tips and tricks to produce results fast! You'll learn all there is to know on using icon fonts to make stunning graphics in Adobe Illustrator.

This is a great process to be familiar with because it helps you produce tangible designs when you are pressed for time, or when you just want to produce a proof of concept, fast.

So join the course and let me show you how I use icon fonts to be a faster designer.

Course Details:

  • Start using font awesome in your designs
  • Get to know the glyph panel
  • Learn how to easily locate icons in the cheat sheet
  • Make your workflow even faster by grouping icons
  • Learn about the pros and cons of icon font alternatives to font awesome
